TERMINAL CHAT SERVER

Compile:
Run the command "make". Ensure that you are in this project's directory.

USAGE:
1.	If you are hosting the server, start it by typing the command "./server"
2.	In a new terminal window, navigate to this project's directory, then type "./client"
3.	Enter your display name (Maximum of 32 characters!)
4. 	Begin typing your messages! Your messages will be sent to all other clients.

COMMANDS:
The client supports commands to control the program.

\q - Quits the client and closes the server connection

BUGS:
-	Typing more than 32 characters for the username will send any characters after the 32nd character as a message.
